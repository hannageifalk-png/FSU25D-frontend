# Visual Studio Code keyboard commands:

Note! Cmd is for MAC users. Ctrl is equivalent of cmd, for PC users 

alt + shift + down       		=> Copy/paste code below
cmd + d 	           		    => Multi select identical code
alt + up/down             		=> Move code up/down
alt+shift+mouse pointer  	    => multi cursor
alt+mouse pointer 		        => Another type of multi cursor

cmd + s			                => Save changes
cmd + shift + 7		            => Toggle block comment
cmd + f			                => Search in specific file
cmd + z			                => Undo changes
cmd + shift + z			        => Take back the undone changes
